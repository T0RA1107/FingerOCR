# 指文字認識
指に貼ったシールの軌跡を文字として認識し、認識した文字列に対応したエフェクトが出る。

## 操作方法
### 起動方法
以下を実行
```
python main.py
```

### 各種コマンド
`q` : プログラム終了\
`s` : スクリーンショットを保存(エフェクトのかかった画像を保存可能)\
`r` : エフェクトの強制終了\
`w` : 押している間指文字を認識(未実装)\
`i` : 指文字を認識(未実装)
